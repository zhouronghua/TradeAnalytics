## 检查截图完整性

运行以下命令检查截图是否完整：

```bash
check_screenshots.bat
```

## 图片规范

- **格式**: PNG（推荐）
- **尺寸**: 800-1400像素宽
- **清晰度**: 保持高清，避免压缩
- **隐私**: 遮挡敏感信息（Webhook key等）
