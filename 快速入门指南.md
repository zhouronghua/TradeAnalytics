## 软件是什么？

TradeAnalytics 是一款**全自动的股票成交量分析系统**，帮助您：

- 🔍 **自动发现**成交量暴涨的股票（5倍以上）
- 📊 **可视化展示**K线图和技术指标
- 📱 **微信推送**分析结果到手机
- 🤖 **定时执行**每天自动分析，无需操作

**适合人群**：个人投资者、短线交易者、投资团队

## 10分钟配置指南

### 第1步：安装软件（2分钟）

```bash
# 1. 确保已安装Python 3.8+
python --version

# 2. 进入项目目录
cd E:\code\TradeAnalytics

# 3. 安装依赖
pip install -r requirements.txt
```

### 第2步：首次运行（2分钟）

```bash
# 启动程序
python main.py
```

界面打开后：
1. 点击"下载股票列表" - 等待10秒
2. 点击"下载历史数据" - 开始后台下载（可继续下面步骤）

### 第3步：配置推送（3分钟）

**企业微信配置**（推荐）

1. 手机下载"企业微信"APP
2. 注册企业（个人免费）
3. 创建群聊"股票分析群"
4. 添加群机器人，复制Webhook
5. 编辑 `config/config.ini`：

```ini
[Notification]
enabled = true
push_type = qywechat
qywechat_webhook = 你的Webhook粘贴这里
```

6. 测试：
```bash
python test_push.py
```

### 第4步：配置定时任务（3分钟）

1. Win+R 输入：`taskschd.msc`
2. 创建基本任务
3. 每天 15:30 执行
4. 启动程序：`run_analysis.bat`
5. 完成！

### 第5步：等待第一次分析

```
今天 15:30，系统自动：
1. 下载最新数据
2. 分析全市场股票
3. 推送结果到微信

您只需查看推送结果即可！
```

## 常见问题快速解答

### Q1: 为什么分析结果是0？

**答**：可能是配置太严格

**解决**：
```ini
# 修改 config/config.ini
[Analysis]
ma_period = 20  # 从120改为20
```

然后重新运行程序。

详细诊断：
```bash
python diagnose_analysis.py
```

### Q2: 数据在哪里？

**答**：
- 股票列表：`data/stocks/stock_list.csv`
- 历史数据：`data/daily/*.csv`（每只股票一个文件）
- 分析结果：`data/results/volume_surge_*.csv`

### Q3: 如何只分析部分股票？

**答**：手动删除不关注的股票数据文件

```bash
# 只保留关注的股票
# 删除data/daily/目录下不需要的csv文件
```

**答**：
```bash

# 3. 查看错误

```

## 关键提醒

### 数据时效性

BaoStock是**T+1数据**：
```
今天: 2026-02-25
最新数据: 2026-02-24（昨天）

这是正常的！
```

建议在**交易日晚上或第二天早上**运行分析。

### 均线周期选择

| 配置 | 数据要求 | 适合场景 |
|------|----------|----------|
| ma_period = 20 | 需要30天数据 | 短线、新手 |
| ma_period = 60 | 需要90天数据 | 波段、中线 |
| ma_period = 120 | 需要150天数据 | 长线、专业 |

**建议新手使用 `ma_period = 20`**

### 筛选结果数量

- 5-20只：正常范围，质量较高
- 20-50只：市场活跃，机会较多  
- 50只以上：可能配置过宽松
- 0只：配置过严格或市场平淡

**建议调整配置，使结果在5-20只之间。**
