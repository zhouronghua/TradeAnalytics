# TradeAnalytics 软件功能介绍

## 软件定位

TradeAnalytics 是一款面向个人投资者的**股票成交量异动分析系统**，通过自动化技术帮助用户快速发现市场中成交量突然放大的股票，提供投资决策参考。

---

## 主要功能模块

### 1. 数据管理模块

![主界面数据管理](screenshots/main_interface.png)

**功能列表**：
- 自动下载A股全市场股票列表
- 批量下载历史K线数据
- 智能增量更新

**数据覆盖**：

### 2. 成交量分析模块

![分析过程](screenshots/volume1.png)

**分析算法**：
```
成交量倍数 = 当日成交量 / 前7日平均成交量

筛选条件：
1. 成交量倍数 ≥ 5.0（可配置）
2. 收盘价 > N日均线（可配置）

同时满足 → 加入结果集
```

**分析结果**：

![分析结果](screenshots/volume2.png)

- 按成交量倍数降序排列
- 显示完整技术指标
- 支持导出CSV格式
- 自动保存历史记录

## 技术特点

### 算法优势

1. **双重验证**
   - 成交量异动
   - 价格趋势确认
   - 减少假信号

2. **动态均线**
   - 根据数据天数自动调整
   - 120日 → 60日 → 30日 → 10日
   - 保证分析的连续性

3. **历史对比**
   - 7日滚动平均作为基准
   - 避免单日波动影响
   - 提高信号可靠性

### 性能优化

1. **多线程下载**
   - 并行下载多只股票
   - 充分利用网络带宽
   - 大幅缩短等待时间

2. **增量更新**
   - 只下载缺失数据
   - 避免重复下载
   - 节省时间和流量

3. **结果缓存**
   - 分析结果自动保存
   - 历史数据可查询
   - 避免重复计算

### 容错设计

1. **数据容错**
   - 自动处理缺失值
   - 异常数据过滤
   - 降级策略（数据不足时）

2. **网络容错**
   - 自动重试机制
   - 失败股票记录
   - 支持断点续传

3. **运行容错**
   - 异常捕获和记录
   - 不中断整体流程
   - 详细错误日志

## 实战示例

### 示例1：发现暴涨股票

**2026-02-25 分析结果**

![示例结果](screenshots/volume4.png)

**股票：002733 雄韬股份**
- 日期：2026-02-25
- 收盘价：21.38元
- 成交量：30,350,815
- 7日平均：4,067,295
- **成交量倍数：7.46x**
- 价格高于均线：5.33%

**分析**：
- 成交量显著放大（7.46倍）
- 价格突破60日均线
- 处于相对低位
- 属于底部启动信号

**后续观察**：
- 是否连续放量
- 价格能否站稳均线
- 是否有基本面支撑

### 示例2：识别假突破

**某股票案例**
- 成交量：8.5倍（符合）
- 收盘价：50元
- 均线：48元（符合）
- 但价格已上涨100%（高位）

**判断**：
- 虽然符合条件
- 但已经涨幅较大
- 可能是高位放量出货
- 需要谨慎

**教训**：
- 不能只看指标
- 要看价格位置
- 要看K线形态
- 综合判断风险

## 配置优化方案

### 方案1：保守型（质量优先）

```ini
[Analysis]
ma_period = 60
volume_ratio_threshold = 10.0
min_history_days = 150
```

**特点**：
- 信号少但质量高
- 假信号少
- 适合稳健投资者

### 方案2：均衡型（推荐）

```ini
[Analysis]
ma_period = 20
volume_ratio_threshold = 5.0
min_history_days = 30
```

**特点**：
- 信号数量适中
- 质量可控
- 适合大多数用户

### 方案3：激进型（机会优先）

```ini
[Analysis]
ma_period = 10
volume_ratio_threshold = 3.0
min_history_days = 30
```

**特点**：
- 信号多
- 机会多
- 需要自行筛选
