## 按使用场景查找

### 场景1：我是新手，第一次使用

推荐阅读顺序：
1. `快速入门指南.md` - 快速了解
2. `VIP_GROUP_QUICKSTART.md` - 配置推送
3. `WINDOWS_SCHEDULER_SUMMARY.md` - 配置定时任务
4. 开始使用！

### 场景2：遇到0结果问题

推荐阅读顺序：
1. `DIAGNOSIS_ZERO_RESULTS.md` - 诊断问题
2. `VOLUME_ANALYSIS_FIX.md` - 了解原理
3. 运行 `diagnose_analysis.py` - 诊断脚本
4. 修改 `config/config.ini` - 调整配置

### 场景3：想深入了解所有功能

推荐阅读顺序：
1. `软件功能介绍.md` - 功能概览
2. `TradeAnalytics使用手册.md` - 完整手册
3. `软件使用说明.md` - 详细说明
4. `README.md` - 项目主文档

### 场景4：配置微信推送

推荐阅读顺序：
1. `PUSH_METHODS_COMPARISON.md` - 了解各种方式
2. `VIP_GROUP_QUICKSTART.md` - 快速配置企业微信
3. 运行 `test_push.py` - 测试推送
4. `VIP_GROUP_PUSH_SETUP.md` - 深入了解

### 场景5：配置定时任务

推荐阅读顺序：
1. `WINDOWS_SCHEDULER_SUMMARY.md` - 快速配置
2. 运行 `check_task.bat` - 验证任务
3. `WINDOWS_SCHEDULER_SETUP.md` - 详细配置

### 场景6：遇到特定问题

查找问题诊断章节：
- 分析结果为0 → `DIAGNOSIS_ZERO_RESULTS.md`
- 数据下载失败 → `BAOSTOCK_FIX.md`
- 股票名称错误 → `STOCK_NAME_FIX.md`
- 推送失败 → 运行 `test_push.py`
- 定时任务不执行 → 运行 `check_task.bat`
