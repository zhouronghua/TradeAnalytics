# VIP群组推送 - 3分钟快速配置

## 效果

每天15:30，您的企业微信VIP群会自动收到股票分析结果。

## 配置步骤

### 1. 创建企业微信VIP群（1分钟）

**还没有企业微信？**

1. 访问 https://work.weixin.qq.com/ 注册（个人免费）
2. 下载"企业微信"APP
3. 创建一个群，命名"股票VIP群"

**已有企业微信？**

直接创建一个VIP群即可。

### 2. 添加群机器人（1分钟）

1. 群聊中点右上角 **"..."**
2. 选择 **"群机器人"** → **"添加机器人"**
3. 复制 **Webhook地址**（保存好！）

### 3. 配置程序（1分钟）

编辑 `config/config.ini`：

```ini
[Notification]
enabled = true
push_type = qywechat
qywechat_webhook = 你刚才复制的Webhook粘贴到这里
```

**示例**：
```ini
[Notification]
enabled = true
push_type = qywechat
qywechat_webhook = https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=abc123def456
```

保存文件（`Ctrl+S`）

### 4. 测试推送（30秒）

```bash
python test_push.py
```

✅ **成功！** VIP群里会收到测试消息。

## 常见问题

### Q: 需要付费吗？
**完全免费！** 企业微信对企业用户完全免费，个人注册也免费。

### Q: 推送次数有限制吗？
**无限制！** 想推送多少次都可以。

### Q: 个人可以注册企业微信吗？
**可以！** 个人可以免费注册，创建自己的"企业"。

### Q: 群消息会通知所有成员吗？
**会的**。群机器人发送消息时，所有群成员都会收到通知。

如果不想打扰，可以设置：
- 群聊设置 → 消息免打扰
- 但能在群里看到消息

### Q: 如何测试推送？
```bash
python test_push.py
```

### Q: 推送失败怎么办？

1. 检查Webhook地址是否完整
2. 检查网络连接
3. 查看日志：`logs/notification.log`
4. 确认机器人未被删除
